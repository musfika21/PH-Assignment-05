Question-1:What are the different ways to select an element in the DOM?

There are several ways to select an element in DOM. Here's the common methods:
1.By ID: document.getElementById("myId")
2.By Class: document.getElementByClassName("myClass")
3.By Tag: document.getElementByTagName("div")
4.CSS Selector (First match): document.querySelector(".myClass")
5.CSS Selector (All matches): document.querySelectorAll(".myClass")

Question-3:What is event delegation in the DOM>

Event delegation is a technique where a parent element handles events for its child elements, even if they are added dynamically.
1.Example:

            document.getElementById("parent").addEventListener("click",function(event){

                 if(event.target.matches(".child")){

                     console.log("child clicked:", event.target);
                }
            });

Question-4:What is event bubbling in the DOM?

Event Bubbling is a concept in the DOM (Document Object Model). It happens when an element receives an event, and that event bubbles up (or you can say is transmitted or propagated) to its parent and ancestor elements in the DOM tree until it gets to the root element.

Question-5:How do you create , add and remove elements using javascript?

1.Create an Element
by document.createElement() we can create a new element
let newDiv = document.createElement("div");
newDiv.textContent = "Hello, World!";
2.Add an Element
document.body.appendChild(newDiv);
3.Remove an Element
let parent = document.getElementById("parent");
let child = document.getElementById("child");
parent.removeChild(child);
